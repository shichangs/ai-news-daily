## 二、AI Agent

**🔥 MCP 正式捐赠 Linux 基金会，Agentic AI Foundation 成立**
Anthropic 将模型上下文协议（MCP）捐赠至 Linux 基金会旗下 AAIF，OpenAI、Block、Google、微软、AWS、Cloudflare、Bloomberg 等联合加入。全球公开 MCP Server 已超 **10,000 个**，ChatGPT、Cursor、Gemini、VS Code 等主流产品均已支持。MCP Dev Summit 定于 4 月 2-3 日在纽约举办。（来源：[Linux Foundation](https://www.linuxfoundation.org/press/linux-foundation-announces-the-formation-of-the-agentic-ai-foundation)）

**企业级 Agent 部署：安全与复杂度成主要瓶颈**
最新调研显示，提示注入（Prompt Injection）和工具污染（Tool Poisoning）已成为 MCP 系统的主要安全风险，认证管控和访问权限管理难度持续阻碍企业大规模部署 Agent。（来源：[Help Net Security](https://www.helpnetsecurity.com/2026/02/24/ai-agents-business-processes-security-complexity/)）

**Gnani.ai × Razorpay 发布 Agent 支付平台**
首个支持 AI Agent 在同一会话内发起并完成 UPI 支付的平台正式上线，结合 Razorpay MCP Server 实现端到端支付 Agent 闭环。（来源：[Business Standard](https://www.business-standard.com/technology/tech-news/gnani-ai-razorpay-launch-agentic-ai-platform-for-in-call-upi-payments-126022600639_1.html)）

**SoundHound 在 MWC 2026 发布零售 Agent**
SoundHound 展示其次世代 Agent 平台，已被数百家大型企业部署，支持语音、短信、网页、邮件、社交、车载等多渠道协同。（来源：[GlobeNewswire](https://www.globenewswire.com/news-release/2026/02/24/3243197/0/en/mwc-2026-soundhound-ai-launches-sales-assist-agent-bringing-real-time-agentic-ai-to-the-retail-sales-floor.html)）

**Airia 成首个完整支持 MCP Apps 的企业 AI 平台**
支持在 AI 对话中直接嵌入交互式仪表盘、表单和可视化图表。（来源：[GlobeNewswire](https://www.globenewswire.com/news-release/2026/02/12/3237135/0/en/Airia-Delivers-First-Enterprise-Support-for-MCP-Apps-Enabling-Interactive-AI-Experiences-at-Scale)）

## 三、AI 产品更新

### 对话与助手类

**🔥 Claude Code 发布 Scheduled Tasks 功能**
Anthropic 在 Cowork 中正式推出 Scheduled Tasks，用户可创建定时任务，让 Claude 定期自动执行工作，无需人工触发——这是 Claude 在常规对话之外首次具备自主定时执行能力。同步推出的还有 Claude Desktop 新版"自定义"栏目，将技能（Skills）、插件（Plugins）和连接器（Connectors）统一管理。（来源：[Claude Help Center](https://support.claude.com/en/articles/13345190-get-started-with-cowork)）

**🔥 ChatGPT Operator 升级为统一 Agent 系统 ChatGPT Agent**
OpenAI 将 Operator 整合升级为 ChatGPT Agent，先向 Pro 用户开放，随后推广至 Plus 和 Team 用户。同步推出 Lockdown Mode（高安全场景）和 Elevated Risk 标签，标注高风险功能。（来源：[Releasebot](https://releasebot.io/updates/openai/chatgpt)）

**ChatGPT Deep Research 全面升级**
支持自定义目标网站、更广泛信息来源，全新 UI 支持创建/编辑研究计划并实时追踪进度。上下文窗口扩展至 256k Token（输入 128k + 输出 128k）。（来源：[adsx.com](https://www.adsx.com/blog/chatgpt-updates-february-2026)）

**ChatGPT 支持单次上传 20 个文件，响应视觉化升级**
文件上传数量上限从 10 个提升至 **20 个**；日常问答回答增加视觉扫描元素，重要人物、地点、产品自动高亮。（来源：[OpenAI Help Center](https://help.openai.com/en/articles/6825453-chatgpt-release-notes)）

**🔥 Gemini 3 Deep Think 发布，ARC-AGI-2 得分 84.6%**
Google 于 2 月 12 日发布 Gemini 3 Deep Think，推理能力大幅领先；2 月 19 日进一步推出 Gemini 3.1 Pro。新增 Personal Intelligence 个性化功能、Chrome 自动浏览和 Veo 3.1 视频生成。（来源：[The Agency Journal](https://theagencyjournal.com/geminis-february-2026-updates-deep-think-personal-intelligence-and-what-actually-changes-for-you/)）

**Perplexity 放弃广告模式，全面转向订阅制**
2 月 18 日 Perplexity 宣布彻底放弃广告收入，转向纯订阅和企业服务。同步发布 Perplexity Computer，将多种 AI 能力整合为统一操作系统。新增 Claude Sonnet 4.6 和 Gemini 3.1 Pro 支持、企业级 Memory 功能。（来源：[Neowin](https://www.neowin.net/news/perplexity-computer-unifies-different-ai-capabilities-into-a-single-system/)）
