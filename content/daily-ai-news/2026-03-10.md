## 3. AI 产品更新

**🔥 Anthropic 发布 Claude Code Review：多 Agent 并行代码审查，Claude Code 年化收入破 $25 亿**

Anthropic 于今日正式推出 **Claude Code Review**，这是 Claude Code 的重大新功能，也是 AI 代码生成与质量把关形成闭环的关键里程碑：

- **工作原理**：PR 开启后，系统自动调度多个 AI Agent 并行审查代码，聚焦可能导致生产故障的 bug，按严重程度排序并过滤假阳性；平均每次 PR 审查约 **20 分钟**；
- **内部验证数据**：过去一年 Anthropic 内部人均代码产出提升 **200%**；Code Review 上线后获得实质性评论的代码变更比例从 **16%** 升至 **54%**；
- **定价与范围**：每次调用 **$15–25**，当前仅限 **Claude Teams 和 Enterprise** 用户的 Research Preview；
- **商业背景**：Claude Code 年化运行收入已突破 **$25 亿**，企业订阅数量自年初翻了两倍，核心客户包括 Uber、Salesforce、Accenture；
- **市场背景**：AI 代码生成工具的大规模普及已使代码审查成为工程瓶颈，工程师每日审查的 AI 生成代码量已超出人力上限。
