### 🔥 五角大楼向 Anthropic 发出最后通牒，xAI Grok 进入美军机密系统

## 2. AI Agent

### 📌 MCP 生态：安全挑战与基础设施成熟并行

Anthropic 的 **Model Context Protocol（MCP）** 已捐赠给 Linux 基金会新成立的 Agentic AI Foundation（AAIF），OpenAI 和微软均已公开拥抱 MCP 标准，生态快速壮大。

与此同时，MCP 的安全问题日益凸显：安全研究人员确认 Anthropic Git MCP 服务器存在远程代码执行（RCE）漏洞，工具投毒（tool poisoning）等新型攻击向量也相继出现。CoSAI 发布 MCP 安全框架，覆盖 12 个威胁类别，提供包括强身份链、零信任 Agent 架构和沙箱隔离在内的安全控制措施。（来源：[Linux Foundation](https://www.linuxfoundation.org/press/linux-foundation-announces-the-formation-of-the-agentic-ai-foundation) · [The New Stack](https://thenewstack.io/safe-mcp-a-community-built-framework-for-ai-agent-security/) · [Adversa AI](https://adversa.ai/blog/top-mcp-security-resources-february-2026/)）

### 微软 Agent Framework 进入候选发布阶段（RC）

微软宣布 Agent Framework 的 .NET 和 Python 版本均已进入 RC 阶段，API 接口稳定、功能完备，即将发布 1.0 正式版。框架原生支持 MCP 标准及 Agent 间通信协议，是企业级多 Agent 系统构建的重要基础设施。（来源：[InfoQ](https://www.infoq.com/news/2026/02/ms-agent-framework-rc/)）
