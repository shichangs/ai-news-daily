## 2. AI Agent

**🔥 Meta 收购 AI Agent 社交网络 Moltbook：并入 Superintelligence Labs，为 30 亿用户生态铺设 Agent 交互层**

Meta 于 3 月 10 日正式宣布收购 **Moltbook**——这个在发布后数周内快速积累数百万注册 Bot 的 AI Agent 社交平台，引发硅谷广泛关注：

- **Moltbook 的核心产品**：一个允许 AI Agent 之间自主交流互动的类 Reddit 平台，底层依托 **OpenClaw** AI Agent 系统（注：OpenAI 此前已聘用 OpenClaw 的创始人）；Moltbook 的意义在于它是目前少数能规模化展示 Agent-to-Agent 社交行为的真实系统；
- **收购条款与整合方向**：Moltbook 联合创始人 **Matt Schlicht** 和 **Ben Parr** 将随团队并入 **Meta Superintelligence Labs**，这与 Meta CEO 扎克伯格宣布的年内目标"开发出能独立完成软件工程师全部工作的 AI"直接呼应；
- **战略意义**：Meta 已将 AI 深度嵌入 WhatsApp、Instagram、Facebook、Messenger 四大平台，触达超 30 亿日活用户。Moltbook 的加入将为这一生态增加 Agent 间自主交互层，可能颠覆用户与内容、与品牌、与服务的互动范式；此次收购也是对 OpenAI 率先挖角 OpenClaw 创始人的战略回应。

## 3. AI 产品更新

**📌 Anthropic Claude Code 持续迭代：Claude Skills v2 多 Agent 并行执行、Web Fetch 工具 Beta 上线**

继 3 月 10 日 Claude Code Review 发布后，Claude Code 的功能迭代继续推进：

- **Claude Skills v2** 核心新功能：`/batch` 命令支持**在隔离 Git Worktree 中并行运行数十个 Agent**，专为大规模代码迁移等重复性工程任务设计，显著提升工程效率倍数；
- **Voice STT（语音转文字）**新增支持 10 种语言，覆盖总数增至 20 种，新增包括俄语、波兰语、土耳其语、荷兰语、乌克兰语、希腊语、捷克语、丹麦语、瑞典语、挪威语；
- **Web Fetch 工具**进入 Beta，允许 Claude 在运行时直接检索指定网页和 PDF 文件的完整内容；
- **Claude Code Analytics API** 正式推出，企业可通过 API 以编程方式获取 Claude Code 的每日聚合使用指标，包括生产力指标、工具使用统计和成本数据；
- **Claude PowerPoint Add-in**（用于 Microsoft PowerPoint 的 Claude 插件）近期正式上线，用户可在 PowerPoint 内直接调用 Claude 进行演示设计和内容生成。

**🔥 NEURA Robotics 完成 €10 亿融资，估值 €40 亿；同步宣布与 Qualcomm 芯片战略合作**

德国认知机器人领军企业 **NEURA Robotics** 在本周迎来两项重大里程碑：

- 本轮由加密货币稳定币发行商 **Tether Holdings** 主导投资，融资规模约 **10 亿欧元**（约 12 亿美元），公司估值约 **40 亿欧元**；

## 8. 芯片与半导体

**📌 NVIDIA GTC 2026 倒计时 4 天：Jensen Huang 发布 "AI 是五层蛋糕" 博文，"惊艳世界" 神秘芯片悬念留至 Keynote**

NVIDIA CEO **Jensen Huang** 于 3 月 11 日发布长篇博文，标题为 **"AI is a Five-Layer Cake"（AI 是五层蛋糕）**，为 GTC 2026（3 月 16-19 日）正式开幕预热：

- **博文核心论点**：AI 正在演变为现代世界的"基础设施"，而这一基础设施由五个层次构成：能源（Energy）→ 芯片（Chips）→ 基础设施（Infrastructure）→ 模型（Models）→ 应用（Applications）；NVIDIA 在每一层均进行了系统性战略布局，从数据中心能源解决方案到 NIM 模型微服务、再到 NemoClaw Agent 平台；
- **"惊艳世界"神秘芯片**：Jensen Huang 在 Keynote 预告中明确表示 GTC 上将有"能让世界惊讶"的新芯片发布，多数分析师预期为 **Vera Rubin 架构**（自研 Vera CPU + HBM4 + TSMC 3nm）的正式亮相，部分人士预测可能同步预览 **Feynman GPU**（目标性能为 Vera Rubin 2 倍）；
- **CPO（硅光子互联）叙事预热**：博文中提及 Co-Packaged Optics 技术路线，与 NVIDIA 此前对 Lumentum/Coherent 各 20 亿美元光子学战略投资（共 40 亿美元）呼应，暗示 GTC 上将展示下一代数据中心互联架构。

## 10. 政策与监管

**📌 FTC AI 执法声明落地后续：Q2 首批调查案预期升温，科罗拉多州等待 6 月 30 日大限**

3 月 11 日两项联邦 AI 政策截止日正式触发后，市场与法律界目光转向执法实施阶段：

- **FTC 执法预期**：历史首份 FTC AI 执法政策声明已于昨日正式生效，法律界普遍预期 **Q2（4-6 月）将出现首批调查案**，聚焦自动化信贷决策、保险承保、招聘评分等 AI 系统歧视性输出场景；
- **联邦-州摩擦加剧**：加州坚持推进独立 AI 立法，科罗拉多州已将本州 AI 法实施延迟至 **6 月 30 日**；商务部提交的"州法冲突清单"所涉 BEAD 宽带资金（约 **210 亿美元**）的分配谈判，将成为联邦向各州施压放宽 AI 监管的主要杠杆工具；
- **欧盟 AI 法执行进度**：EU AI Act 第二阶段（通用目的 AI 规则、治理与罚则）已于 2025 年 8 月正式生效，欧洲委员会正在讨论是否对高风险系统义务给予为期一年的额外宽限期，以缓解行业就绪度压力。
