## 3. AI 产品更新

**🔥 Anthropic 发布 Claude Code Review：多 Agent 并行代码审查，每次 PR 费用 $15–25**

Anthropic 于 3 月 10 日发布 **Claude Code Review**，这是 Claude Code 的重大功能扩展，也是 Anthropic 将"AI 代码生成"与"AI 代码质量把关"形成闭环的关键一步：

- **核心机制**：每当 PR（Pull Request）被开启，系统自动调度**多个 AI Agent 并行**对代码进行审查，聚焦于可能导致生产环境故障的 bug，按严重程度排序并过滤假阳性；
- **审查深度**：平均每个 PR 约 **20 分钟**完成审查，专注代码正确性而非代码格式或测试覆盖率等偏好性问题；
- **内部验证数据**：过去一年 Anthropic 内部代码人均产出提升 **200%**；Code Review 上线后，获得实质性评论的代码变更比例从 **16%** 跃升至 **54%**；
- **价格与可用范围**：每次调用 **$15–25**（依 PR 复杂度），目前仅限 **Claude Teams 和 Enterprise** 用户的 Research Preview 阶段。

背景：AI 代码生成工具的大规模普及已导致代码审查成为新瓶颈——工程师每天要审查的 AI 生成代码量已超出人力上限，Claude Code Review 直接切入这一痛点。（来源：[VentureBeat - Claude Code Review](https://venturebeat.com/technology/anthropic-rolls-out-code-review-for-claude-code-as-it-sues-over-pentagon) · [Winbuzzer - Code Review launch](https://winbuzzer.com/2026/03/10/anthropic-claude-code-review-parallel-ai-agents-bugs-security-xcxwbn/) · [Dataconomy - Claude Code Review](https://dataconomy.com/2026/03/10/anthropic-launches-ai-powered-code-review-for-claude-code/)）

## 10. 政策与监管

**🔥 FTC + 商务部双截止日今日到期：美国首份 AI 执法政策声明正式触发，州法冲突清单提交**

今日（3 月 11 日）是特朗普 AI 行政令规定的**两大联邦 AI 政策截止日**，对美国 AI 监管格局具有里程碑意义：

**① FTC 历史首份 AI 执法政策声明（今日截止）**

FTC 主席须在今日前发布政策声明，正式阐明《联邦贸易委员会法》第 5 条（禁止不公平/欺骗性商业行为）如何适用于 AI 模型——据泄露草案，声明覆盖三大场景：AI 生成广告、训练数据同意框架、自动化决策透明度（信贷评分、保险承保、雇佣决策）；**执法可即时生效**，无需新立专项 AI 法律。预计 Q2 将出现首批基于该声明的 FTC 调查案（焦点可能是"AI 审核了您的贷款申请"等决策场景）。

据法律学者指出，该声明还可能对加利福尼亚州、科罗拉多州、伊利诺伊州等要求 AI 模型修改输出内容以减少偏见的州法产生**联邦优先（preemption）**效果——实质上以现行消费者保护法对部分 AI 州法形成挑战。

**② 商务部州法冲突清单（今日截止）**

商务部长须提交各州被认定为阻碍 AI 创新的法律清单，作为联邦通过 **BEAD 宽带资金**（约 **210 亿美元**）向各州施压、推动松绑地方 AI 监管的前置步骤。
