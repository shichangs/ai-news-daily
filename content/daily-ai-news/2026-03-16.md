## 1. 主要 AI Labs 动态

**📌 Anthropic 发布 Agent 能力三项新 API：代码执行工具、MCP 连接器、Files API 同步上线**

NVIDIA GTC 开幕当日，Anthropic 在 Anthropic API 上同步推出三项重要更新，提升开发者构建 Agent 应用的能力：

- **代码执行工具（Code Execution Tool）**：Claude 可在安全沙盒环境中直接编写并运行代码，无需外部服务器；
- **MCP 连接器（MCP Connector）**：Claude 可通过标准化 MCP 协议直接连接外部工具、数据库和 API，实现一行代码级别的工具集成；
- **Files API**：允许在多次 API 调用之间持久化存储文件（最大 32 MB/文件，500 MB/组织），解决了 Agent 跨会话数据传递的核心痛点；
- **1 小时 Prompt 缓存**：上下文缓存时长从 5 分钟延长至 **1 小时**，大幅降低长对话和 Agent 任务的调用成本。

这是 Anthropic API 在 Agent 基础设施方面最全面的一次更新，被认为是对 OpenAI Realtime API 和 Google Agent Framework 的直接竞争响应。（来源：[Anthropic - New agent capabilities API](https://www.anthropic.com/news/agent-capabilities-api) · [Releasebot - Anthropic March 2026](https://releasebot.io/updates/anthropic)）

## 8. 芯片与半导体

**🔥 NVIDIA GTC Keynote：Vera Rubin 平台正式亮相，$1 万亿订单目标，7 芯片 5 种机架，CUDA 20 周年**

今日 Jensen Huang Keynote 芯片层面核心公告（历时 **2 小时+**，CUDA **20 周年**纪念日）：

**Vera Rubin 平台组成**（7 芯片 + 5 种 Rack）：
- **Rubin GPU**：集成 HBM4，单卡内存带宽超 **10 TB/s**；NVL72 推理系统相比 Blackwell 每瓦推理吞吐提升 **10 倍**，每 token 成本降低 **90%**；
- **Vera CPU**：NVIDIA 自研 ARM 架构第三代，将 NVIDIA 从单纯 GPU 公司正式转型为"全栈计算平台"公司；
- **Groq 3 LPU + LPX 机架**：借助 2026 年 1 月 $200 亿收购 Groq 后的技术整合，LPX 机架搭载 **256 块 LPU**，较 Rubin GPU 可将推理 token/watt 再提升 **35 倍**；
- 配套硬件：NVLink 6 Switch（互联）、ConnectX-9 SuperNIC（网络）、BlueField-4 DPU（数据处理）、Spectrum-6 Ethernet Switch（以太网）；

**商业信号**：Jensen Huang 上调 Blackwell + Vera Rubin 联合采购订单预测至 **$1 万亿**（2027 年前），较此前 $5,000 亿预测翻倍；

**IGX Thor 正式 GA**：工业级边缘物理 AI 计算平台（用于工厂、仓库、医院等边缘场景）全面可用；

**Vera Rubin Space-1**：专为卫星和轨道数据中心设计的特殊版本，延伸 AI 推理至太空场景。
