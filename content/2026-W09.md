## 本周概览

本周是 2026 年迄今为止 AI 行业最动荡、也最具历史分量的一周。**OpenAI 完成 1100 亿美元史上最大单笔私募融资**，估值达 7300 亿美元，Anthropic 此前也已完成 300 亿美元 G 轮；两家顶级 AI 实验室合计募资逾 1400 亿美元，助推 2 月全球 AI 融资总额突破 **1950 亿美元**，创单月历史纪录。与此同时，**Anthropic 与美国五角大楼的正面冲突**成为本周最受关注的事件：CEO Dario Amodei 拒绝开放 Claude 用于大规模监控和自主武器，国防部随即切断合作，特朗普将 Anthropic 定性为"觉醒"公司——这场博弈正在重塑整个 AI 行业的军事应用边界。芯片层面，英伟达 **Vera Rubin** 系统首次曝光，Meta 同日与英伟达、AMD 签下合计逾 600 亿美元的芯片大单，算力军备竞赛持续升温。在 Agent 生态方面，MCP 正式捐赠给 Linux 基金会，成为"AI 的 USB-C"行业标准，但随即暴露出 8000 余台服务器无认证裸奔的严峻安全隐患。

## 分板块周报

### 1. 主要 AI Labs 动态

**融资格局重构**

Anthropic 于 2 月 12 日率先完成 300 亿美元 G 轮融资（估值 3800 亿美元，2025 年 ARR 接近 70 亿，2026 目标 150 亿），随后 OpenAI 于 2 月 27 日完成 1100 亿美元融资（估值 7300 亿，亚马逊 500 亿 + 英伟达 300 亿 + 软银 300 亿），两家公司合计本月募资逾 1400 亿美元，正式进入另一个量级的资本竞争。

**最大争议：Anthropic 军事红线**

这是本周最重要的事件。五角大楼要求 Anthropic 解除 Claude 在监控公民和完全自主武器上的限制，CEO Dario Amodei 拒绝，国防部援引国家安全法切断合作，特朗普将 Anthropic 定性为"觉醒"公司并命联邦机构停用其服务。OpenAI Sam Altman 公开支持 Anthropic 立场，同时宣布 OpenAI 以纳入安全协议为条件与军方合作。200 余名谷歌和 OpenAI 员工联署声援。xAI Grok 则正式进入美军机密系统，成为 Claude 的直接竞争者。

**模型发布**

Anthropic 的 Claude Sonnet 4.6 于 2 月 17 日正式发布，官方称以 Sonnet 价格实现 Opus 级性能，盲测约 70% 的情况下用户偏好优于 Sonnet 4.5；Claude Opus 4.6 同步推出，支持 100 万 Token 上下文（Beta）与 12.8 万 Token 输出，新增 Agent Teams 与 effort 控制。Claude Sonnet 5（SWE-Bench Verified 82.1%，首破 80%）于 2 月 3 日已上线。

OpenAI 发布 GPT-5.2（400K Token 上下文，SWE-bench 69%，支持完整视频处理，定价 $1.75/$14）；另外开源 gpt-oss-120b（性能接近 o3/o4-mini，可单卡运行）和 20B 轻量版（可在消费级笔记本运行）。

Google 于 2 月 12 日发布 Gemini 3 Deep Think（ARC-AGI-2 得分 84.6%），2 月 19 日推出 Gemini 3.1 Pro（ARC-AGI-2 77.1%），均在推理能力上大幅领先前代。Google DeepMind 还发布 Genie 3 世界模型，可实时生成用户可交互的 3D 世界（720p、24fps）。

**其他重要动态**：Anthropic 收购计算机操作 AI 创业公司 Vercept（融资约 5000 万美元），持续强化 Claude Agent 能力；Anthropic 指控 DeepSeek 用数千个虚假账户窃取训练数据；Snowflake 与 OpenAI 签署 2 亿美元战略合作协议。

### 2. AI Agent

**MCP 成为行业标准**

Anthropic 将模型上下文协议（MCP）捐赠给 Linux 基金会旗下新成立的 Agentic AI Foundation（AAIF），OpenAI、Google、微软、AWS、Cloudflare、Bloomberg 等联合加入，全球公开 MCP Server 已超 10,000 个。MCP 正被业界称为"AI 的 USB-C"，ChatGPT、Cursor、Gemini、VS Code 等主流产品均已支持。MCP Dev Summit 定于 4 月 2–3 日在纽约举办。

**安全危机浮出水面**

MCP 月 SDK 下载量从 2024 年 11 月的 10 万暴增至 9700 万，但规模扩张远超安全治理能力：安全研究人员扫描发现超 8000 台 MCP 服务器裸奔于公网（无任何身份验证），Anthropic Git MCP Server 存在 RCE 漏洞，工具污染（Tool Poisoning）和提示注入等新型攻击向量相继出现。CoSAI 已发布覆盖 12 个威胁类别的 MCP 安全框架，提供零信任架构和沙箱隔离等控制措施。

**企业生态拓展**

微软 Agent Framework 的 .NET 和 Python 版本进入 RC 阶段（原生支持 MCP），Atlassian 在 Jira 中引入 AI Agent 协作（人机同看板任务分配），Agent 工作流编排初创公司 Trace 完成 300 万美元种子轮融资。Gnani.ai × Razorpay 发布首个支持 Agent 在同一会话内完成 UPI 支付的平台。

### 3. AI 产品更新

**Claude Code 全面升级**

本周 Claude Code 推出三项重大功能：Scheduled Tasks（定时任务，Claude 可无需人工触发自主执行定期工作）、Remote Control（跨设备远程控制本地 Agent）、--worktree 模式（隔离 Git Worktree 并行开发），以及 background: true 后台任务支持。GitHub Copilot 将 Claude Code 纳入 Agent HQ 多 Agent 平台，VS Code 1.109 支持 Claude、Codex、Copilot 并行运行。

**ChatGPT 更新**

Operator 升级为 ChatGPT Agent（先 Pro 后 Plus/Team），新增 Lockdown Mode；Deep Research 支持自定义目标网站和可编辑研究计划；单次文件上传上限从 10 个提升至 20 个；上下文窗口扩至 256k Token；代码块升级为可交互模式。

**编程工具竞争加剧**

Cursor 推出 2.0 版（最多 8 个并行 Agent、云端 Agent 独立启动 VM 并自动提交 PR、Composer 1.5 速度提升 4 倍）；Windsurf Wave 13 带来 Arena Mode 并排对比和 Plan Mode；Cline v3.58 支持原生 Subagent 和 CLI 2.0 无头 CI/CD 模式。"Vibe Coding"（自然语言编程）被 MIT Technology Review 评选为 2026 年突破性技术。

**Perplexity 转型**

Perplexity 2 月 18 日宣布彻底放弃广告收入，全面转向订阅制，并发布 Perplexity Computer 将多种 AI 能力整合为统一操作系统。

**图像/视频生成**

Midjourney V7 正式发布（新增文字转视频与 3D 功能，处理速度更快、写实感更强）。字节 Seedance 2.0 定位专业影视制作，生成的"Tom Cruise vs Brad Pitt 打架视频"病毒式传播，成为本月最受关注的 AI 视频内容。

### 4. 中国 AI 产业

**资本与上市**

智谱 AI 成全球首家独立上市大模型公司，港股市值突破 511 亿港元，同步开源 GLM-5（对标 Claude Opus 4.5 编程能力）。月之暗面（Kimi）完成 7 亿美元融资，估值突破 100 亿美元。

**模型密集发布**

字节跳动春节前后密集发布：Doubao 2.0（复杂推理达 ChatGPT/Gemini 同等水平）、Seeddream 5.0（图像生成）、Seedance 2.0（视频生成，病毒式传播）。阿里 Qwen 3.5 开源，显存降 60%、推理提效 19 倍。DeepSeek GitHub 仓库出现"MODEL1"新架构标识，被广泛解读为 V4 技术基础。Anthropic 指控 DeepSeek 用虚假账户抓取训练数据，国际法律争议持续。

**应用层 DAU 大战**

字节豆包用 3 年达到 7871 万 DAU；阿里千问仅用 82 天（借助 30 亿元免单活动）从不足 1000 万暴涨至 7352 万 DAU，两大平台正面交锋。行业从底层模型比拼转向消费者服务渗透，阿里已测试"对话即购物"功能。

**资本支出规模**

字节跳动 2026 年 AI 资本支出高达 1600 亿元（一半用于芯片），阿里计划 2025–2027 年共投入 3800 亿元用于云和 AI 基础设施。阿里云以 35.8% 市场份额大幅领跑中国 AI 云市场（字节火山引擎 14.8%、华为云 13.1%）。中国春节 AI 红包大战：腾讯、阿里、百度、字节合计投入超 45 亿元争夺流量入口。

### 6. 自动驾驶
